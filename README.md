# Prime AC Field App

Deploy-ready Next.js app for phone and tablet use.

## What is included
- Prime AC branded technician workflow
- QR review screen
- Customer intake
- Equipment screen
- Inspection screen
- Readings screen
- Estimate builder
- Closeout screen

## Run locally
1. Install Node.js 18 or newer
2. In this folder run:
   npm install
   npm run dev
3. Open http://localhost:3000

## Deploy to Vercel
1. Create a Vercel account
2. Upload this folder or import it from GitHub
3. Click Deploy
4. Open the live link on your phone
5. In Safari or Chrome tap Add to Home Screen

## Notes
- This is a front-end prototype and does not yet save data to a database.
- A developer can next add PDF generation, signatures, login, and customer history.
